hubroid by eddieringle

The Awesome GitHub app for Android.

Hubroid allows you to check up on your favorite GitHub repositories and users from your Android device. Search repositories and users, view specific information about each: read commit logs, view commit diffs, browse trees, activity feeds, issues, and much more!
